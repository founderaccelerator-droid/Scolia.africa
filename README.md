# SCOLIA

SCOLIA is an EdTech platform that helps young people learn digital skills, build real projects, and earn income online.

## 🚀 Mission

Learn → Build → Earn

## 📚 Features

- Digital skills learning system
- AI learning assistant
- Project builder system
- Freelance marketplace (future)
- Community challenges

## 🧠 Tech Stack

- Ruby on Rails
- PostgreSQL
- Redis (future scaling)
- OpenAI API

## 🎯 Goal

To become Africa’s leading skill-to-income platform.
